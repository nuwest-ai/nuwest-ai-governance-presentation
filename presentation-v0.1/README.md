## Quick Start

```bash
# Navigate to this directory
cd /Users/home/Documents/nuwest/2026/presentations/CISOSummit2025/

# Start local server
python3 -m http.server 8000

# Open in browser
open http://localhost:8000
```

## Keyboard Shortcuts

- **Arrow Keys** - Navigate slides
- **S** - Speaker notes
- **F** - Fullscreen
- **B** - Blackout
- **?** - Help

## PDF Export

1. Add `?print-pdf` to URL: `http://localhost:8000?print-pdf`
2. Print to PDF (Cmd+P)
3. Set landscape, no margins
