### Design Guidelines

**Color Palette:**
- Primary: Yellow #FFD300 (use for emphasis, key elements, warnings)
- Secondary: Black #000000 (use for outlines, text, backgrounds)
- Accent: White #FFFFFF (use for backgrounds, contrast)

**Typography (if text in images):**
- Headers: IBM Plex Sans Bold
- Body: Inconsolata Regular
- Logo/brand: Chakra Petch Semi-Bold

**Icon Style:**
- Line weight: 2-3px for consistency
- Corners: Slightly rounded (not sharp, not too round)
- Fill: Generally use outline style with yellow fill
- Size: Icons should scale well from 80px to 400px

**Aesthetic Principles:**
- Functional over decorative
- Warning signage aesthetic
- High contrast for visibility
- Minimal detail
- Clear hierarchy

## Usage in Slides

### How to Add Graphics to Markdown Slides

**Method 1: Inline image**
```markdown
![Description](assets/graphics/slide_09_pattern_escalation.svg)
```

**Method 2: Background image**
```markdown
<!-- .slide: data-background="assets/graphics/slide_09_pattern_escalation.png" data-background-size="contain" -->
```

**Method 3: Positioned image with CSS class**
```markdown
<img src="assets/graphics/slide_26_four_metrics.svg" class="icon-large" alt="Four Alignment Metrics">
```

### CSS Classes Available (from theme.css)

- `.icon-inline` - Small inline icons (1.2em)
- `.icon-large` - Large standalone icons (4em)
- `.icon-grid` - Grid layout for multiple icons
- `.stat-box` - Styled box for statistics
- `.visual-separator` - Yellow/black separator line
